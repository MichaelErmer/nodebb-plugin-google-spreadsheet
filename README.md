# NodeBB Google Spreadsheet Plugin

This NodeBB plugin allows users to embed Spreadsheets inline in posts.

## Installation

    npm install nodebb-plugin-google-spreadsheet

## Usage

    Paste the URL into a post, the system will do the rest.
